<h2>🌟 About The Project</h2>

<p>
Every movie review tells a story. Some express excitement, others disappointment.
This project leverages <b>Natural Language Processing (NLP)</b> and <b>Machine Learning</b>
to automatically determine whether an IMDb movie review conveys a
<b>Positive 😊</b> or <b>Negative 😞</b> sentiment.
</p>

<p>
The system processes raw text, extracts meaningful patterns, and predicts sentiment with high accuracy,
making it useful for sentiment analysis, recommendation systems, and customer feedback analytics.
</p>
